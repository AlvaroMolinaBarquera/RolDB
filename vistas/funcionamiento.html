¿Como Funciona la API?

RolDB está diseñada sobre un pequeño servidor json creado por typicode, este servidor json está diseñado para realizar pruebas pero ofrece todos lo que normalmente ofrece una API. La documentación del servidor está aqui pero tambien se explican las más comunes a continuación.

En primer lugar vale la pena indicar que hay 2 direcciones principales:

  • /jdr Que es donde se encuentran los juegos de rol.
  • /books Que es donde se encuentran los libros.

Para acceder a una busqueda general se emplea ?q= + Valor a buscar. Ejemplo

  • /jdr?q=Mago
  • /books?q=basico

Tambien es posible obtener resultados por pagina para evitar cargar un monton de elementos a la vez

  • /jdr?_page=2 devolveria la pagina 2.

Para buscar por un valor especifico basta se usa la siguiente formula

  • /books?title=El Desollador devuelve el libro solo si el titulo coincide 100%
  • /jdr?name=anima_like devuelve todos los juegos que contengan anima
